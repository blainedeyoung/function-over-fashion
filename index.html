The set up for this site pretty much the same as Fashion over Function. The big addition is jQuery. It wasn't included previously because I just started learning it. And I'm already wondering how I ever got along without it. I'm not really happy with the way the accordion turned out. It seemed like a good idea in my head, but when I saw it implemented I decided it's cumbersome and redundant. I'm going to turn it over in my mind for a while and see if I can't come up with something better to replace it with or some way to use it better. Anyway as noted in the gallery, this site is a mock-up, a front-end stub. The CSS is themed Bootstrap 5.0.1. All of the photos on this page have linked popovers which give just credit to the good folks at unsplash.com. If this were an actual working website, I'd replace those popovers with links to product pages. The icons are from fontawesome. The fonts come from Google fonts. The code for this site is available on github. I hope you enjoyed the show. This link will take you back to the gallery. Best regards.
